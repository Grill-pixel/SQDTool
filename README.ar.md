# SQDTool – لوحة إدارة التشكيلة

SQDTool هو تطبيق مكتبي (Tkinter) لإدارة قائمة اللاعبين وتعديلها بسهولة، ثم توليد تقرير PDF مرتب. الواجهة منظمة على شكل بطاقات حديثة وتوفر **مبدّل ثيمات** بعدة ألوان.

اللغات: [Français](README.md) | [English](README.en.md) | [الرخصة](LICENSE)

## المزايا الأساسية

- **تعديل اسم النادي والموسم** بدون قيم ثابتة.
- **تحرير كامل للتشكيلة** (إضافة/تعديل/حذف).
- **تصدير PDF** حسب المراكز.
- **واجهة حديثة** مع 12 ثيمًا مختلفًا.
- **حفظ الإعدادات تلقائيًا**.

## المتطلبات

- Python 3.9+
- اتصال بالإنترنت (فقط لتثبيت `fpdf2` تلقائيًا عند أول تشغيل)

## التثبيت والتشغيل

```bash
git clone <your-repo>
cd SQDTool
python marseille.pyw
```

> في أول تشغيل، سيقوم التطبيق بتثبيت `fpdf2` إذا لم يكن موجودًا.

## الميزات بالتفصيل

### 1) إعدادات النادي
من بطاقة **« Paramètres du club »** يمكنك تعديل:

- **اسم النادي**
- **الموسم**
- **الثيم**

اضغط **Appliquer** لحفظ التعديلات وتحديث عنوان البرنامج ولوحة الحالة وعنوان الـ PDF.

### 2) مبدّل الثيمات
القائمة المنسدلة تحتوي على 12 لوحة ألوان (مثل: Azur & Or، Océan Profond، Lavande Moderne). يتم تطبيق الثيم مباشرة ويتم حفظه تلقائيًا.

### 3) محرّر التشكيلة
اضغط **« Éditer la composition »** لفتح محرر اللاعبين:

- إضافة/تعديل/حذف اللاعبين.
- الحفظ في ملف `composition.json`.
- واجهة شبيهة بالجدول مع نموذج بيانات.

### 4) تصدير PDF
اضغط **« Générer le PDF »** للتصدير:

- تجميع تلقائي حسب المركز.
- يستخدم اسم النادي والموسم الحاليين في عنوان التقرير.
- الحفظ داخل مجلد الإخراج المحدد (افتراضيًا `PDFs/`).

## الملفات والحفظ

- `composition.json` : بيانات التشكيلة.
- `settings.json` : اسم النادي + الموسم + الثيم.
- `om_program_logs.txt` : سجلات التشغيل.
- `PDFs/` : مجلد ملفات الـ PDF.

## التخصيص

لإضافة ثيم جديد، عدّل القاموس `themes` داخل `marseille.pyw`:

```python
"My Theme": {
    "bg": "#...",
    "card": "#...",
    "primary": "#...",
    "accent": "#...",
    "text": "#...",
    "muted": "#...",
    "button": "#...",
    "button_text": "#...",
}
```

## حل المشاكل

- **مشكلة تثبيت FPDF**: نفّذ الأمر `pip install fpdf2` يدويًا.
- **عدم العثور على PDF**: تحقّق من مجلد الإخراج والصلاحيات.
- **مسارات الخطوط على macOS/Linux**: عدّل مسارات الخط داخل `generate_pdf_file()`.
