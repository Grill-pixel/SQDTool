# SQDTool — لوحة حديثة لإدارة التشكيلة

![Python](https://img.shields.io/badge/Python-3.9%2B-3776AB?logo=python&logoColor=white)
![UI](https://img.shields.io/badge/UI-Tkinter-0F172A)
![الترخيص](https://img.shields.io/badge/License-MIT-10B981)

**SQDTool** تطبيق سطح مكتب (Tkinter) لإدارة قائمة اللاعبين، ترتيب التشكيلة على الملعب، وتصدير ملفات PDF بشكل نظيف وجاهز للمشاركة. يجمع بين محرّر تشكيلة مرن، عرض ملعب تفاعلي، ونظام ثيمات عصري.

## الميزات

- **إدارة كاملة للتشكيلة**: إضافة، تعديل، حذف اللاعبين بسرعة.
- **توزيع تكتيكي تفاعلي**: وضع اللاعبين وتحريك المراكز يدويًا.
- **PDF للتشكيلة**: مُرتّب حسب المراكز.
- **PDF للتوزيع**: نسخة مطابقة لعرض الملعب على الشاشة.
- **ثيمات حديثة**: 12 لوحة ألوان مع حفظ تلقائي.
- **إعدادات محفوظة**: اسم النادي، الموسم، والثيم.

---

## انطلاقة سريعة

1. شغّل التطبيق: `python marseille.pyw`.
2. عدّل القائمة عبر **تعديل التشكيلة**.
3. رتّب الفريق عبر **التوزيع**.
4. صدّر ملفات PDF للتشكيلة والملعب.

---

## التثبيت

```bash
git clone <your-repo>
cd SQDTool
python marseille.pyw
```

> في أول تشغيل، يتم تثبيت **fpdf2** تلقائيًا عند الحاجة. تصدير التوزيع يثبت **Pillow** إذا لزم الأمر.

## طريقة الاستخدام

### إعدادات النادي
في **إعدادات النادي** يمكنك تعديل:

- **اسم النادي**
- **الموسم**
- **الثيم**

اضغط **تطبيق** لتحديث الواجهة والملفات المصدّرة.

### محرّر التشكيلة
افتح **تعديل التشكيلة** من أجل:

- إضافة لاعب (الاسم، المركز، العمر، الجنسية، الحالة، خيار الشراء).
- تعديل سطر موجود.
- حذف المدخلات المحددة.
- حفظ التشكيلة بصيغة JSON.

### التوزيع (الملعب)
عرض **التوزيع** يسمح ببناء شكل تكتيكي على نصف ملعب:

- **وضع اللاعبين**: اختر لاعبًا ثم اضغط على مركز لوضعه.
- **تحريك المراكز**: اسحب المراكز لتعديل المسافات.
- **تحديث القائمة**: يعيد تحميل اللاعبين عند التغيير.
- **إعادة ضبط**: إلغاء التوزيع أو إعادة ضبط المراكز.

### تصدير ملفات PDF

- **PDF للتشكيلة**: زر **توليد PDF** (مرتّب حسب المراكز).
- **PDF للتوزيع**: زر **تصدير إلى PDF** داخل نافذة التوزيع.

ملف PDF للتوزيع هو **لقطة مطابقة** للعرض الحالي على الملعب (المراكز والأسماء والتنسيق).

## الملفات والحفظ

- `composition.json` (أو `effectifOM.json`): بيانات التشكيلة.
- `settings.json`: اسم النادي، الموسم، الثيم.
- `om_program_logs.txt`: سجلات التشغيل.
- `PDFs/`: ملفات PDF المصدّرة.

## استكشاف الأخطاء

- **خطأ في تصدير PDF**: تأكد من وجود اتصال بالإنترنت لتثبيت `fpdf2` و `pillow` لأول مرة.
- **عدم ظهور ملف PDF**: تحقق من المجلد الذي اخترته عند التصدير.
- **قصّ النص داخل الملعب**: كبّر النافذة قبل التصدير.
